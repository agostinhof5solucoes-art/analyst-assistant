# analyst-assistant
Projetada para otimizar e padronizar o fluxo de atendimento técnico das equipes de suporte N1 e N2. O sistema nasceu de uma dor crônica do mercado de automação comercial: a lentidão no diagnóstico de erros complexos em sistemas de ERP e (PDV), o que eleva o tempo de espera do cliente e gera perda de conhecimento técnico dentro das software houses.
